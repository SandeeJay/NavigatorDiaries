# NavigatorDiaries

This is a mobile application for travel enthusiasts who wish to document their travels with ease. The app provides auto geo-tagging, itinerary planning, and text-to-speech capabilities to enhance the user experience.

This project was developed as a part of the Final Year Project at the University of Bedfordshire.

### Technologies Used
The application was developed using the following technologies:

* C#
* Xamarin
* Xamarin.Android
* SQLite
* XAML

### Usage
Once the application is installed, you can use it to:

* Create a travel itinerary
* Add entries with auto geo-tagging
* Add photos and notes
* Use text-to-speech to listen to your journal entries (Story Mode)
